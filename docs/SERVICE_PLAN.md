# 물린김에 분석함 서비스 기획서

## 1. 서비스 정의

`물린김에 분석함`은 한국/미국 주식과 주요 섹터의 흐름을 매일 리서치 관점으로 정리해 보여주는 Apps in Toss 미니앱이다.

서비스의 핵심은 매수/매도 지시가 아니라, 사용자가 오늘 시장을 이해하고 이미 물린 포지션을 어떻게 점검할지 정리하도록 돕는 것이다.

### 포지셔닝

- 주식 추천 앱이 아니라 AI 리서치 리포트 큐레이션 앱으로 운영한다.
- 모든 종목 표현은 `아직 살아있는 흐름`, `관찰 필요`, `과열 주의`, `깨지는 조건 확인` 같은 리서치 언어를 사용한다.
- `무조건 사라`, `반드시 오른다`, `수익 보장`, `손실 보전`처럼 투자 판단을 단정하거나 보장하는 표현은 금지한다.
- 사용자가 보기 쉬운 형태로 요약하되, 리스크와 확신도를 항상 함께 보여준다.

### 핵심 사용자 가치

- 긴 경제 뉴스와 종목 이슈를 한 번에 읽지 않아도 된다.
- 한국/미국 시장을 같은 기준으로 비교할 수 있다.
- 지금 버틸 근거가 남은 흐름과 깨지는 조건을 빠르게 구분할 수 있다.
- Codex 자동화로 생성된 일관된 리서치를 앱에서 매일 확인할 수 있다.

## 2. 콘텐츠 구조

사용자에게 정보를 한 번에 모두 보여주지 않고, 아래 순서로 나눠 제공한다.

1. 10초 판단: 오늘의 한 줄 결론
2. 시장 온도와 업데이트 시각
3. 왜 그런가: 시장 온도 판단 이유
4. 아직 살아있는 흐름 3개
5. 한국/미국 시장 요약
6. 더 깊게 볼 섹터와 종목 흐름
7. 물리기 쉬운 포인트
8. 부자/기관/스마트머니 참고 흐름
9. 이번 주 관찰 우선순위
10. 조심해야 할 리스크
11. 다음에 확인할 체크포인트

### 첫 화면 구성

첫 화면은 사용자가 10초 안에 오늘 시장 분위기를 이해하도록 구성한다.

- 앱 이름: 물린김에 분석함
- 업데이트 상태: `오늘 08:40 업데이트` 또는 `주말 요약`
- 오늘의 한 줄 결론
- 시장 온도: `공격`, `중립`, `보수`, `관망`
- 아직 살아있는 흐름 요약
- 깨지는 조건 한 줄
- 피드백 버튼: `도움됐어요`, `애매해요`
- 토스 배너 광고 영역

## 3. 업데이트 운영 정책

### 권장 업데이트 시간

기본 업데이트 시간은 한국 시간 기준 오전 8시 40분으로 한다.

이유:

- 한국 장 시작 전 사용자가 빠르게 확인할 수 있다.
- 미국 장 마감 이후 주요 흐름을 반영할 수 있다.
- 환율, 금리, 유가, 미국 선물 흐름을 함께 요약하기에 적절하다.

### 한 줄 결론 표현 방식

아침 리포트에서는 하루 결과를 단정하지 않는다.

좋은 예:

- `오늘은 공격적 매수보다 관찰 우선. AI 전력 인프라와 반도체는 강하지만, 환율 부담은 체크가 필요합니다.`
- `미국 기술주 흐름은 우호적이나 한국 장에서는 외국인 수급 확인 전까지 분할 관찰이 유리합니다.`
- `오늘 신규 진입은 보수적으로 보고, 이미 오른 테마는 추격보다 눌림 확인이 필요합니다.`

나쁜 예:

- `오늘은 이 종목을 사면 됩니다.`
- `오늘 반드시 오를 종목입니다.`
- `10만원이면 A 종목을 매수하세요.`

### 리포트 종류

| 상황 | 리포트 타입 | 설명 |
|---|---|---|
| 평일 오전 | `daily_pre_market` | 장 시작 전 리서치. 오늘의 관찰 방향 중심 |
| 평일 장 마감 후 | `daily_review` | 당일 움직임 리뷰. MVP 이후 선택 |
| 주말 | `weekly_watch` | 지난주 리뷰와 다음주 체크포인트 |
| 휴장일 | `market_closed` | 신규 리포트 없음. 다음 거래일 안내 |

MVP에서는 `daily_pre_market`과 `weekly_watch`만 구현한다.

## 4. 광고 정책

광고는 토스 광고만 사용한다. 외부 광고 네트워크는 사용하지 않는다.

사용 가능한 광고 형태:

- 배너 광고
- 전면형 광고
- 리워드 광고

### MVP 광고 배치

- 홈 하단에 배너 광고를 배치한다.
- 상세 리포트 중간 또는 하단에 배너 광고를 배치한다.
- 전면형 광고는 앱 진입 직후에는 사용하지 않는다.
- 리워드 광고는 `TOP 10 전체 보기`, `지난 리포트 더 보기` 같은 부가 기능에만 검토한다.

### 광고 UX 원칙

- 광고가 콘텐츠보다 많아 보이면 안 된다.
- 광고가 투자 행동 버튼처럼 보이면 안 된다.
- 리포트 핵심 결론은 광고 없이 먼저 보여준다.
- 광고 때문에 리스크 문구가 가려지면 안 된다.

### 구현 메모

현재 `/reference` 폴더에는 토스 포인트 프로모션과 공유 리워드 문서만 있다. 전면형/배너/리워드 광고 API 문서는 별도로 확보한 뒤 구현한다.

## 5. 피드백 정책

MVP에서는 댓글과 좋아요/싫어요를 구현하지 않는다.

대신 리포트 단위로 아래 두 가지 피드백만 받는다.

- 도움됐어요
- 애매해요

### 비로그인 정책

- 토스 로그인은 MVP에서 사용하지 않는다.
- 비로그인 상태로 리포트를 볼 수 있다.
- 피드백 중복 방지는 가능하면 Apps in Toss의 anonymous key 또는 서버 발급 익명 식별자를 사용한다.
- 댓글, 관심종목 저장, 알림, 리워드형 기능이 필요해질 때 토스 로그인을 검토한다.

## 6. 데이터 생성 및 저장 구조

로컬 Codex 자동화를 통해 매일 리포트를 생성하고, 특정 API를 통해 서버/DB에 저장한 뒤 앱에서 조회한다.

### 기본 흐름

```
Codex 자동화
  -> 시장 리서치 생성
  -> JSON 형태로 정규화
  -> API 저장
  -> 앱에서 최신 리포트 조회
```

### 추천 API

| 메서드 | 경로 | 설명 |
|---|---|---|
| `GET` | `/reports/latest` | 최신 리포트 조회 |
| `GET` | `/reports/:date` | 특정 날짜 리포트 조회 |
| `POST` | `/reports` | Codex 자동화가 리포트 저장 |
| `POST` | `/reports/:id/feedback` | 도움됐어요/애매해요 저장 |

### 리포트 JSON 초안

```json
{
  "id": "2026-05-11-daily-pre-market",
  "date": "2026-05-11",
  "type": "daily_pre_market",
  "publishedAt": "2026-05-11T08:40:00+09:00",
  "marketTemperature": "보수",
  "oneLineConclusion": "오늘은 공격적 매수보다 관찰 우선입니다.",
  "confidence": "medium",
  "top3": [],
  "koreaSummary": [],
  "usSummary": [],
  "hotStocks": [],
  "watchStocks": [],
  "riskStocks": [],
  "longTermStocks": [],
  "smartMoneySignals": [],
  "top10": [],
  "risks": [],

    "helpful": 0,
    "unclear": 0
  }
}
```

## 7. 주말 및 휴장일 정책

주말에는 평일처럼 `오늘 투자할 종목`을 억지로 만들지 않는다.

주말 콘텐츠:

- 지난주 시장 요약
- 다음주 체크포인트
- 실적 발표 일정
- 금리, 환율, 유가, 미국 선물 관찰 포인트
- 관심 섹터 변화

앱 안내 문구:

`주말과 주요 휴장일에는 신규 장전 리포트가 발행되지 않을 수 있어요. 대신 다음 거래일을 준비하는 체크포인트를 정리해드려요.`

## 8. 투자 표현 가드레일

모든 리포트와 UI는 아래 원칙을 따른다.

- 투자 추천이 아니라 리서치 참고 자료라고 명시한다.
- 매수/매도 명령형 표현을 사용하지 않는다.
- 금액별 포트폴리오는 `예시 시나리오`로만 제공한다.
- `오늘 투자하지 않는 것도 선택지`를 명확히 제공한다.
- 과열 종목에는 반드시 리스크를 함께 표시한다.
- 확신이 낮은 내용은 확신 낮다고 표시한다.
- 한국 주식과 미국 주식을 비교하되, 특정 국가/섹터 편향을 단정하지 않는다.

### 금액별 문구 원칙

기존 프롬프트의 `10만원, 30만원, 50만원 기준`은 유지하되, 앱에서는 다음처럼 표현한다.

- `10만원 예시 관찰안`
- `30만원 예시 분산안`
- `50만원 예시 분산안`

단, 시장이 불리하면 단호하게 아래처럼 표시한다.

`오늘은 신규 투자보다 관망이 더 낫다고 판단됩니다. 관심 종목은 정리하되, 진입은 주요 지표 확인 이후로 미루는 편이 좋겠습니다.`

## 9. MVP 범위

### 1차 MVP

- 최신 리포트 조회
- TDS 기반 홈 화면
- 리포트 섹션 나눠보기
- 도움됐어요/애매해요 피드백
- 토스 배너 광고 자리
- 주말/휴장일 안내
- 리서치 참고 고지

### 2차

- 과거 리포트 목록
- 종목별 상세 근거 화면
- 리워드 광고 기반 추가 콘텐츠
- 공유 기능

### 3차

- 토스 로그인
- 관심종목 저장
- 댓글 또는 한줄 의견
- 알림/구독형 기능
